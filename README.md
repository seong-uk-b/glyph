# Glyph (글리프)

> 다국어 문자/어휘 학습 앱 — React + TypeScript

## 로고

```
       ●            ← 골드 점 (宀의 점)
    ╭─────╮         ← 아치 (宀 갓머리)
       │
    ───┼───         ← 십자 (子)
       │
       ╰─           ← 갈고리
```

**디자인 컨셉:** 한자 **字**(글자 자)를 기하학적으로 추상화

| 요소 | 표현 | 의미 |
|------|------|------|
| 宀 아치 | 그라데이션 곡선 (레드→퍼플→블루) | 문자의 지붕, 학습의 틀 |
| 子 십자 + 갈고리 | 세로획 + 가로획 | 문자의 뼈대, 쓰기의 흐름 |
| 골드 점 | 금색 원형 | 시작점, 핵심 포인트 |

## 주요 기능

### 일본어
- **히라가나 / 카타카나 퀴즈** — 직접 입력(타이핑) 또는 4지선다
- **가나 문자표** — 오십음도 + 탁음/반탁음 차트
- **JLPT 단어장** — N5 ~ N3 단어 퀴즈 (뜻→단어 / 단어→뜻)

### 한국어
- **한글 자모 퀴즈** — 자음/모음 → 로마자 발음 맞추기
- **한글 문자표** — 기본/쌍자음, 기본/복합모음 차트
- **한글 조합 퀴즈** — 조합된 글자(가, 난 등) → 로마자 발음 맞추기
- **한국어 단어장** — 기본 단어 65개 (영어/일본어 뜻)

### 공통
- **다국어 TTS** — 일본어/한국어/영어 음성 지원 (Web Speech API)
- **다국어 UI** — 한국어 / 영어 / 日本語 전환 지원
- **학습 언어 자동 전환** — 학습 언어와 UI 언어 충돌 시 자동 조정
- **Atom Dark 테마** — 깔끔한 다크 테마 디자인

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm start

# 프로덕션 빌드
npm run build

# GitHub Pages 배포
npm run deploy
```

[http://localhost:3000](http://localhost:3000) 에서 확인할 수 있습니다.

## 게임 흐름

```
[일본어]
홈 → 설정 (문자 종류, 모드, 행 선택) → 가나 게임 → 결과
홈 → 단어 설정 (레벨, 모드, 문제 수) → 단어 게임 → 결과

[한국어]
홈 → 한글 설정 (자모 카테고리) → 한글 퀴즈 → 결과
홈 → 조합 설정 (받침 포함 여부) → 조합 퀴즈 → 결과
홈 → 단어 설정 (모드, 문제 수) → 단어 게임 → 결과
```

| CRA | 빌드 도구 |
| GitHub Pages | 배포 |

## 프로젝트 구조

```
src/
  data/           # 문자/단어 데이터 (hiragana, katakana, hangul, words/, korean-words)
  hooks/          # 게임 상태 관리 (useGameStateBase + 각 게임별 훅)
  utils/          # 게임 로직 (shuffle, 정답 판별, 선택지 생성, TTS)
  styles/         # 테마 & 애니메이션
  i18n/           # 다국어 지원 (en/ko/ja)
  components/
    common/       # Button, ToggleGroup, CheckboxGrid
    layout/       # Header, Footer
    screens/      # 일본어/한국어 각 게임별 Setup, Game, Result, Chart
    game/         # CharacterDisplay, TypingMode, MultipleChoiceMode, SpeakButton
```
